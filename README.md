# Dskyz / DAFE Open-Source Collection

A curated archive of open-source TradingView Pine Script indicators, libraries, and strategies originally published by **Dskyz (DAFE) Trading Systems**.

> **Preservation notice:** TradingView publications can be removed or made private at any time. This repository preserves point-in-time snapshots of publicly released scripts so that the community can continue to study and reference them. All original authorship credit belongs to **Dskyz (DAFE) Trading Systems**.
